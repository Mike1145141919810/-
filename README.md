
# 班级教学随机点名系统（Web）

一个面向课堂教学场景的轻量级随机点名网页工具。  
项目主打“开箱即用 + 视觉简约高级 + 操作低门槛”，适合教师在课堂中快速、公平地进行互动点名。

## 项目亮点

- **本地名单读取**：支持读取同目录 TXT 名单文件（每行一个姓名）
- **动态抽取效果**：点击开始后姓名高速闪动，点击结束立即锁定结果
- **多人同时抽取**：可设置一次抽取人数，满足小组回答场景
- **默认去重机制**：已抽中的姓名不会在本轮重复出现，抽完后可自动进入新一轮
- **中文兼容优化**：兼容常见中文编码文本，降低乱码概率
- **课堂友好设计**：深蓝偏黑背景、居中大字号显示，投屏可读性强

## 适用场景

- 课堂提问随机点名
- 小组代表随机抽取
- 班会互动与游戏化教学
- 需要“公平、快速、可视化”抽签的教学活动

## 使用说明（简要）

1. 准备名单 TXT 文件（每行一个姓名）
2. 打开网页并读取名单
3. 设置抽取人数
4. 点击“开始抽取”与“结束抽取”完成点名

## 设计目标

通过简单稳定的交互和清晰的视觉反馈，减少课堂组织成本，让教师把注意力集中在教学本身。
